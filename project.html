Football Player's Market Value Prediction

Area:

Sports Analytics using Artificial Neural Network ANN

In recent years, neural networks and deep learning have become popular methods for predicting the market value of football players.
In this project Neural Network was implemented on a large dataset of player's value of year 2021 that was used to train models that can accurately predict the value of a player based on various factors such as age, height, weight, stamina, strength, skill moves, work rate, performance, team etc.
With the help of this method, football clubs can make better decisions when it comes to buying and selling players.
